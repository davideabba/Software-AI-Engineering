Una libreria JS che consente di potenziare il proprio HTML con funzionalità tipiche di:
- AJAX
- CSS Transitions
- WebSockets
- Server Sent Events
iniettandole direttamente nel codice HTML.

Il vantaggio è la capacità di poter costruire siti più moderni e responsivi senza dover dipendere da framework frontend esterni come React o Vue.

**Esempi di casi d'uso:**
> - Lanciare una richiesta HTTP con tag diversi da `<a>` e `<form>`
> - Usare trigger eventi diversi da `click` e `submit`
> - Avere altri metodi oltre a `GET` e `POST`
> - Aggiornare o caricare elementi singoli della pagina senza dover ricaricare la pagina intera
